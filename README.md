# MarkLines

This extension emulates XCode's `// MARK: -` comment decorations.

![A MARK comment separating one section of code from the next.](https://raw.githubusercontent.com/everdrone/mark-meta-comments/main/.github/media/screenshot.png)

# Installation

Install it from the [VSCode Marketplace](https://marketplace.visualstudio.com/items?itemName=jrpat.marklines)

# Usage


- `// MARK: - Foo` to get a line above the comment
- `// MARK: Foo -` to get a line below the comment
- `// MARK: - Foo -` to get a line above and below the comment
